# NFO Subtitle Quality Export Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Write XML NFO sidecar files alongside subtitle files containing provider/score/backend metadata — expert feature, off by default (`auto_nfo_export = false`).

**Architecture:** A standalone `backend/nfo_export.py` module writes `{subtitle_path}.nfo` XML files. Two integration points: (1) `translator.py` after each translation save, (2) `wanted_search.py` after each provider download. Both call `maybe_write_nfo()` which short-circuits when disabled. Two API endpoints for manual export. Frontend exposes the setting (advanced) and an Export NFO button per sidecar badge.

**Tech Stack:** Python stdlib `xml.etree.ElementTree`, existing `config.py` Pydantic settings, existing `is_safe_path()` guard, existing sidecar badge pattern in `SeriesDetail.tsx`

**Worktree:** `.worktrees/feat/v0.27.0-nfo-export`

- [ ] **Step 3: Implement `backend/nfo_export.py`**

  ```python
  """NFO sidecar export — writes Kodi/Jellyfin-compatible XML metadata alongside subtitle files.

  This is an expert feature disabled by default (auto_nfo_export = false).
  """
  import logging
  import xml.etree.ElementTree as ET
  from datetime import datetime

  logger = logging.getLogger(__name__)

  _FIELDS = [
      "provider",
      "source_language",
      "target_language",
      "score",
      "translation_backend",
      "bleu_score",
      "downloaded_at",
      "sublarr_version",
  ]


  def _is_enabled() -> bool:
      try:
          from config import get_settings
          return bool(getattr(get_settings(), "auto_nfo_export", False))
      except Exception:
          return False


  def _get_version() -> str:
      try:
          from version import __version__
          return __version__
      except Exception:
          return ""


  def write_nfo(subtitle_path: str, metadata: dict) -> None:
      """Write an NFO XML sidecar to ``{subtitle_path}.nfo``.

      Errors are logged and swallowed — NFO writing must never interrupt the
      subtitle pipeline.
      """
      nfo_path = subtitle_path + ".nfo"
      try:
          root = ET.Element("subtitle")
          meta = dict(metadata)
          meta.setdefault("sublarr_version", _get_version())
          meta.setdefault("downloaded_at", datetime.utcnow().strftime("%Y-%m-%dT%H:%M:%S"))
          for field in _FIELDS:
              val = meta.get(field)
              ET.SubElement(root, field).text = "" if val is None else str(val)
          tree = ET.ElementTree(root)
          ET.indent(tree, space="  ")
          with open(nfo_path, "wb") as f:
              tree.write(f, xml_declaration=True, encoding="utf-8")
          logger.debug("Wrote NFO sidecar: %s", nfo_path)
      except Exception as exc:
          logger.warning("Failed to write NFO sidecar %s: %s", nfo_path, exc)


  def maybe_write_nfo(subtitle_path: str, metadata: dict) -> None:
      """Write NFO only when auto_nfo_export is enabled in settings."""
      if not _is_enabled():
          return
      write_nfo(subtitle_path, metadata)
  ```

- [ ] **Step 3: Implement `backend/routes/nfo.py`**

  ```python
  """NFO export API routes."""
  import logging
  import os

  from flask import Blueprint, jsonify, request

  import nfo_export
  from security_utils import is_safe_path

  logger = logging.getLogger(__name__)
  nfo_bp = Blueprint("nfo", __name__, url_prefix="/api/v1")


  def _get_media_path() -> str:
      from config import get_settings
      return getattr(get_settings(), "media_path", "/media")


  @nfo_bp.post("/subtitles/export-nfo")
  def export_nfo_single():
      path = request.args.get("path", "").strip()
      if not path:
          return jsonify({"error": "path is required"}), 400
      if not is_safe_path(path, _get_media_path()):
          return jsonify({"error": "Forbidden"}), 403
      if not os.path.isfile(path):
          return jsonify({"error": "File not found"}), 404

      nfo_export.write_nfo(path, {})
      return jsonify({"status": "ok", "nfo_path": path + ".nfo"})


  @nfo_bp.post("/series/<int:series_id>/subtitles/export-nfo")
  def export_nfo_series(series_id: int):
      from database import get_db, _db_lock

      with _db_lock:
          db = get_db()
          rows = db.execute(
              "SELECT path FROM subtitle_files WHERE series_id = ? AND path IS NOT NULL",
              (series_id,),
          ).fetchall()

      media_path = _get_media_path()
      written = 0
      errors = 0
      for (path,) in rows:
          if not path or not is_safe_path(path, media_path):
              continue
          if not os.path.isfile(path):
              continue
          try:
              nfo_export.write_nfo(path, {})
              written += 1
          except Exception as exc:
              logger.warning("NFO export failed for %s: %s", path, exc)
              errors += 1

      return jsonify({"status": "ok", "written": written, "errors": errors})
  ```

  **Note:** The `subtitle_files` table may not exist — check `database.py` for the actual table/column names for series subtitle paths. If a different table is used, adjust the query. Common alternatives: query `sidecar_subtitles` or scan via the library endpoint pattern.
