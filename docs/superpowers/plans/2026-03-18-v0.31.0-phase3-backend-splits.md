# v0.31.0 Phase 3 — Backend File Splits

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Split 8 oversized backend files (800–2,921 lines) into focused sub-modules without changing any behavior, preserving all external import paths.

**Architecture:** Each `routes/X.py` becomes a `routes/X/` package; the package `__init__.py` creates the Flask Blueprint and imports submodules that register routes on it. `from routes.X import bp` in `routes/__init__.py` is unchanged. Non-route modules (`translator.py`, `wanted_search.py`) become packages (`translator/__init__.py`, `wanted_search/__init__.py`) that re-export everything — all external `from translator import ...` and `from wanted_search import ...` imports continue to work. This is a pure structural refactor: no behavior changes, no API changes, no new features.

**Tech Stack:** Python 3.11, Flask Blueprints, threading.Lock (shared state module pattern)

## Blueprint Package Pattern (used for all route splits)

```python
# routes/hooks/__init__.py
from flask import Blueprint

bp = Blueprint("hooks", __name__, url_prefix="/api/v1")

# Import submodules AFTER bp is defined — they register routes on bp
from routes.hooks import events, crud, webhooks, logs, scoring  # noqa: E402, F401
```

```python
# routes/hooks/events.py
from routes.hooks import bp  # bp is already defined when this runs

@bp.route("/events/catalog", methods=["GET"])
def event_catalog():
    ...
```

This works because when Python imports `routes.hooks`, it:
1. Creates the module object and adds it to `sys.modules`
2. Executes `__init__.py`, creating `bp`
3. Imports submodules — when they do `from routes.hooks import bp`, Python finds the partially-initialized module in `sys.modules` and gets `bp` (already defined in step 2)

```python
"""Shared helpers and constants for tools routes."""

import os
import shutil

from security_utils import is_safe_path

SUPPORTED_FORMATS = {"srt", "ass", "ssa", "vtt"}
PYSUBS2_EXT = {"srt": "srt", "ass": "ass", "ssa": "ssa", "vtt": "vtt"}


def _validate_file_path(file_path: str) -> tuple:
    """Validate that file_path exists, is a subtitle, and is under media_path.

    Returns:
        (error_message, status_code) if invalid, (None, abs_path) if valid.
    """
    from config import get_settings, map_path

    if not file_path:
        return ("file_path is required", 400)

    file_path = map_path(file_path)

    s = get_settings()
    if not is_safe_path(file_path, s.media_path):
        return ("file_path must be under the configured media_path", 403)

    abs_path = os.path.realpath(file_path)

    if not os.path.exists(abs_path):
        return (f"File not found: {file_path}", 404)

    ext = os.path.splitext(abs_path)[1].lower()
    if ext not in (".srt", ".ass", ".ssa"):
        return ("Only .srt, .ass, and .ssa files are supported", 400)

    return (None, abs_path)


def _create_backup(file_path: str) -> str:
    """Create a .bak backup of a file before modifying it."""
    base, ext = os.path.splitext(file_path)
    bak_path = f"{base}.bak{ext}"
    shutil.copy2(file_path, bak_path)
    return bak_path
```
